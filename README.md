# Разработка алгоритма обнаружения объектов в MUR IDE

Программа для автономного необитаемого подводного аппарата (АНПА) в симуляторе MUR IDE. Система реализует интеллектуальный поиск и захват подводных объектов в реальном времени с использованием нейросетевого детектора **YOLOv8**[cite: 3].

## Возможности
*   **Универсальный захват:** Обнаружение любых объектов (сферы, цилиндры, маркеры) без жесткой привязки к классам[cite: 3, 4].
*   **Вычисление координат:** Автоматический расчет центроида объекта для последующего наведения аппарата[cite: 3].
*   **Визуализация:** Отрисовка рамок (bounding boxes) и вывод текущего показателя FPS для контроля производительности[cite: 3].
*   **Стабильный вывод:** Использование OpenCV для обхода ошибок совместимости стандартного API симулятора[cite: 3].

## Быстрый старт
1.  Установите **MUR IDE** с официального сайта [murproject.com](https://murproject.com).
2.  Установите необходимые Python-зависимости:
    ```bash
    python -m pip install ultralytics opencv-python numpy
    ```
3.  Разместите скрипт проекта в директории симулятора.
4.  Запустите миссию (клавиша **F5** в MUR IDE).

## Настройка параметров
В основном программном модуле вы можете отрегулировать следующие параметры:

| Параметр | Значение по умолчанию | Описание |
| :--- | :--- | :--- |
| `conf` | 0.3 | Порог уверенности нейросети (чем ниже, тем чувствительнее захват)[cite: 4] |
| `model` | yolov8n.pt | Версия предобученных весов (используется Nano-версия для скорости)[cite: 3] |
| `time.sleep` | 0.01 | Частота обновления цикла обработки[cite: 3] |

## Устранение неполадок
*   **Ошибка `AttributeError`:** Исправлена в текущей версии кода путем перехода на `cv2.imshow`[cite: 3].
*   **Объект не обнаруживается:** Снизьте значение `conf` до **0.2** или проверьте освещенность в симуляторе[cite: 4].
*   **Низкая скорость (лагает):** Убедитесь, что используется модель `yolov8n.pt` (самая быстрая в линейке)[cite: 3].
